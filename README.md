📝 Notes Web App

A simple and modern browser-based notes application built using HTML, CSS, and JavaScript.
It allows users to create, edit, and delete notes with automatic saving using LocalStorage, ensuring notes persist even after refreshing the page.

This project focuses on clean UI design, DOM manipulation, and local data storage without using any external libraries or frameworks.

🚀 Features

✍️ Create new notes

📝 Edit existing notes

🗑️ Delete notes

💾 Persistent storage using LocalStorage

⏱ Automatic timestamp for created notes

🎨 Modern dark UI design

📱 Smooth UI animations for editor and notes list

📂 Dynamic rendering of notes from storage
